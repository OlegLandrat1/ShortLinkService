# Short Link Service / Сервис Сокращения Ссылок

![Java](https://img.shields.io/badge/Java-17%2B-orange)

Сервис сокращения ссылок с возможностью управления лимитом переходов и временем жизни ссылки.

##  Функции

- ✅ **Сокращение URL**: Преобразование длинных ссылок в короткие
- ✅ **Генерация уникальных кодов коротких ссылок для каждого пользователя** Каждый пользователь получает UUID, с помощью которого генерируются уникальные короткие ссылки для одинаковых URL
- ✅ **Лимит переходов**: Настройка максимального количества переходов при создании ссылок
- ✅ **Время жизни (TTL)**: Автоматическое удаление истёкших ссылок
- ✅ **UUID-идентификация**: Работа без авторизации. При запуске сервиса автоматически генерируется UUID пользователя
- ✅ **Открытие в браузере**: Автоматический переход по ссылке через `Desktop.browse()`
- ✅ **Статистика**: Просмотр общего кол-ва созданных ссылок и ссылок пользователя
- ✅ **Уведомления**: Информирование об истечении лимитов и TTL
- ✅ **CLI интерфейс**: Управление через командную строку


## Основные компоненты:

- **ShortLink**: Модель короткой ссылки с метаданными
- **ShortLinkService**: Основная бизнес-логика
  - ***createShortLink***: Генерация уникальных коротких кодов
  - ***clickShortLink***: Переход по ссылке
  - ***deleteLink***: Удаление ссылок
- **NotificationService**: Уведомления пользователей
- **listMyLinks, showStats**: Статистика и просмотр ссылок
- **run**: Обработка CLI команд

## Установка и запуск

### Требования

- Java 17 или выше

### Компиляция

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/ShortLinkService.git
cd ShortLinkService

# омпилировать файл java
javac ShortLinkService.java

# Запустить приложение
java -jar ShortLinkService.jar
```
